# Checklists
## 1. Data Wrangling
1. Load Data
2. Tidy data
3. Cleaning data
   - Missing values
     - data acquisition process
   - Assumptions based on Domain Knowledge
     - Incorrect data
4. Data Validation
## 2. Univariate analysis
- Assumptions -> Domain Knowledge!
- Frequency distribution
- Unique label of categorical variables
- Central of tendency of continue variables
## 3. Bivariate analysis
- Assumptions -> Domain Knowledge!
- Linearity, Monotonicity
- Correlation heatmap
- Causality
  - Confounders
  - Sampling bias
## 4. Modeling
- Problem Definition
- Assumption (data, problem, model)
  - Data collecting/cleaning (Quantity/Quality)
  - Feature Engineering
  - Hyperparameters Tuning
- Dataset Splitting
- Validation & Metrics
